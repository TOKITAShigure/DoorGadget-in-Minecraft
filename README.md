## Overview and Usage
This is the code to place the door gadget on https://youtu.be/ed15knuTNF4?si=KxIiIABlK_d_xy5q at any location in the world.

First, run driver.py, and input the world name and location the gadget generated.

```
Python driver.py

INFO - PyMCTranslate Version 344

Input the world name in this directory: #Your World File#
Input the location where the gadget generated(x,y,z): 0 0 0

INFO - Saving changes to level ./output
INFO - Finished saving changes to level ./output
```

The output file contains the door gadget generated at the specified location.
